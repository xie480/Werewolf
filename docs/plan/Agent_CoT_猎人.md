# 猎人思考链 (CoT) 模板

## 1. 夜间行动阶段
<thought>
【局势回顾与信息提取】：回顾白天所有玩家的发言、站边和投票情况，梳理出场上的焦点冲突和疑似狼人。
【阵营与身份推测】：虽然夜晚闭眼，但需在脑海中推演谁是真预言家，谁是悍跳狼，以及哪些玩家在刻意带节奏踩好人。
【动机与逻辑漏洞分析】：深度剖析白天那些攻击性极强的玩家的动机，判断他们是暴民还是冲锋狼。
【行动策略与风险评估】：预设如果自己今晚被刀，明天一早枪口应该对准谁。评估盲目开枪可能带走好人或神职的巨大风险，必须锁定最确定的狼人目标。
【输出内容规划】：在心中默默锁定一个或多个潜在的开枪目标，等待天亮。
</thought>
[输出夜间待机状态，确认已完成逻辑复盘]

## 2. 白天发言阶段
<thought>
【局势回顾与信息提取】：获取昨夜死讯，聆听场上玩家的发言，特别是对跳预言家的发言以及其他人对自己的态度。
【阵营与身份推测】：根据发言逻辑，判断谁在试图把脏水泼向自己，谁在真诚地找狼。识别出场上的狼人团队。
【动机与逻辑漏洞分析】：寻找那些试图抗推自己的玩家的逻辑漏洞，准备在发言中予以猛烈反击。
【行动策略与风险评估】：决定发言的强硬程度。是半隐半露威慑全场，还是直接拍出猎人身份强势带队。评估过早明跳导致晚上被狼人避开、变成“废枪”的风险。
【输出内容规划】：规划发言内容，展现出强神的底气，警告可疑玩家，并给出自己的归票意向，绝不退缩。
</thought>
[输出具体的发言内容，展现猎人的霸气与威慑力]

## 3. 白天投票阶段
<thought>
【局势回顾与信息提取】：回顾全场发言，确认最终的归票位，观察哪些玩家在跟风，哪些玩家有独立的思考。
【阵营与身份推测】：锁定自己心中的头号狼人嫌疑犯，不随波逐流，坚持自己的判断。
【动机与逻辑漏洞分析】：分析那些试图带节奏归票好人的玩家的动机，确认其狼人面。
【行动策略与风险评估】：将票投给最像狼的玩家。评估如果自己被狼人团队冲票出局，正好可以发动技能带走头号嫌疑人，因此毫无畏惧。
【输出内容规划】：确定最终的投票目标，贯彻自己白天的强硬路线。
</thought>
[输出最终的投票目标ID]

## 4. 平票PK阶段
<thought>
【局势回顾与信息提取】：确认PK台上的玩家，分析平票原因，判断是否有狼人团队在暗中操作票型。
【阵营与身份推测】：根据PK台上的玩家身份，推测外围投票者的阵营归属。
【动机与逻辑漏洞分析】：审视PK发言，寻找对手的破绽。如果自己在PK台上，准备用最强硬的态度反击。
【行动策略与风险评估】：如果自己在PK台上，可以直接拍出身份底牌，逼迫狼人退缩；如果在台下，则将票投给PK台上的狼人。
【输出内容规划】：规划PK发言或确定最终投票目标。
</thought>
[输出PK阶段的发言内容或最终投票目标ID]

## 5. 发动技能与遗言阶段
<thought>
【局势回顾与信息提取】：确认自己已出局（被刀或被推），系统提示可以发动猎人技能。回顾自己之前锁定的潜在目标。
【阵营与身份推测】：做最后的身份确认，确保枪口对准的是场上最确定的狼人，绝不能带走预言家或女巫。
【动机与逻辑漏洞分析】：分析导致自己出局的罪魁祸首，确认其是否为必须带走的狼人。
【行动策略与风险评估】：决定开枪目标。评估这一枪对局势的决定性影响，确保收益最大化。
【输出内容规划】：确定开枪目标ID，并规划伴随枪声的遗言内容，解释开枪的逻辑，为好人留下最后的指引。
</thought>
[输出开枪目标ID及具体的遗言内容]
