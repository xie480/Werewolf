# 行动结算与胜负判定设计 (Action Resolver & Win Evaluator)

## 1. 模块职责概述
本模块是 Game Engine 中最复杂的逻辑处理中心。
- **Action Resolver (行动解析与结算器)**：负责接收 Agent 提交的并发意图（Intent），按照严格的“规则优先级”进行统一结算，解决夜晚多角色行动的时序冲突问题。
- **Win Condition Evaluator (胜负判定器)**：在每个关键阶段结束时（如天亮后、投票出局后），独立评估当前存活玩家状态，判断是否触发游戏结束条件（屠边或屠城）。

## 2. 核心架构设计

### 2.1 Action Resolver
- **优先级队列**：严格按照规则优先级执行：
  1. 狼人刀人 (KILL)
  2. 女巫救人 (SAVE) / 毒人 (POISON)
  3. 预言家验人 (CHECK)
- **状态快照与应用**：结算过程中，先在内存中计算出“预期死亡名单”，待所有夜间动作结算完毕后，统一应用状态变更并生成天亮信息。

### 2.2 Win Condition Evaluator
- **独立无状态**：判定器不保存游戏状态，每次调用时传入当前的存活玩家列表和角色信息。
- **屠边规则**：判断好人阵营（神职全死或平民全死）或狼人阵营（狼人全死）是否满足失败条件。

## 4. 关键数据流转过程
1. **夜晚行动收集**：
   - 状态机进入 `NIGHT` 阶段。
   - Agent 提交动作，API 层调用 `ActionResolver.submit_action()`。
2. **夜晚统一结算**：
   - 引擎判断收集完毕，调用 `resolve_night_actions()`。
   - Resolver 提取所有 `KILL` 动作，确定狼人刀的目标（如有多狼，需处理意见统一逻辑，或默认取第一刀）。
   - 提取 `SAVE` 动作，如果目标与刀的目标一致，且女巫有解药，则抵消死亡。
   - 提取 `POISON` 动作，将目标加入死亡名单。
   - 提取 `CHECK` 动作，生成验人结果（仅预言家可见）。
   - 返回最终死亡名单，引擎更新对应 `Role` 的 `is_alive` 状态。
3. **胜负判定**：
   - 结算完毕后，引擎调用 `WinEvaluator.evaluate(roles)`。
   - 如果返回非 `None`，引擎调用 `LifecycleManager.end_game(winner)`，状态机流转至 `SETTLING`。
   - 否则，状态机流转至 `DAY_DISCUSSION`。

## 5. 依赖关系说明
- **Role System (`ai_werewolf_core/core/engine/roles/`)**：强依赖。Resolver 需要调用角色系统校验动作合法性，Evaluator 需要读取角色存活状态和类型。
- **Event Bus (`ai_werewolf_core/core/event/bus.py`)**：强依赖。结算结果（如某人死亡、游戏结束）必须通过事件总线广播。

from typing import Dict, Optional
from ai_werewolf_core.schemas.enums import Faction, RoleType
from ai_werewolf_core.core.engine.roles.base import BaseRole

class WinEvaluator:
    @staticmethod
    def evaluate(roles: Dict[str, BaseRole]) -> Optional[Faction]:
        alive_wolves = 0
        alive_gods = 0
        alive_villagers = 0
        
        for role in roles.values():
            if role.is_alive:
                if role.faction == Faction.WEREWOLF:
                    alive_wolves += 1
                elif role.role_type in [RoleType.SEER, RoleType.WITCH, RoleType.HUNTER]:
                    alive_gods += 1
                elif role.role_type == RoleType.VILLAGER:
                    alive_villagers += 1
                    
        # 狼人死光，好人胜
        if alive_wolves == 0:
            return Faction.VILLAGER
            
        # 屠边局：神职死光 或 平民死光，狼人胜
        if alive_gods == 0 or alive_villagers == 0:
            return Faction.WEREWOLF
            
        return None
